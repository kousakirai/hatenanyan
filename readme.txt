        概要
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
「AquesTalk10」は、簡単に小型機器への組み込みが出来る音声合成ミドルウェアです。

このライブラリを用いることで、簡単にテキスト情報を音声波形に変換出力することが
可能になります。

本SDKは、AquesTalk10 規則音声合成ライブラリを、アプリケーションに組み込んで使用
するための一連のドキュメントとソフトウェアを含んだものです。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
        評価板の制限と製品版
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
評価板は製品版と異なり、以下の制約が有ります。

	*****************************************************
	「ナ行、マ行」を指定すると、すべて「ヌ」と発声します
	*****************************************************

本ライブラリの実行時に「開発ライセンスキー」を指定することで、上記制約の無い製
品版として動作します(製品版のライブラリは別途用意されていません)。
「開発ライセンスキー」を指定しない場合は、評価版として動作します。

開発ライセンスキーは、開発ライセンスを購入したときに発行されるライセンス証に記
載されています。
